# 사주킹 (Saju King)

**사주킹**은 AI 기반 개인 맞춤형 사주 및 운세 서비스입니다.

## 주요 기능

### 1. 사주 분석 (Saju Analysis)
- 개인의 생년월일시 정보를 바탕으로 정밀한 사주팔자 분석 결과를 제공합니다.
- 오늘의 운세, 신년 운세 등 다양한 운세 정보를 확인할 수 있습니다.

### 2. 궁합 분석 (Compatibility Analysis)
- 연인, 친구, 동료 등과의 사주 궁합을 분석하여 관계에 대한 조언을 제공합니다.
- 서로의 성향 차이와 조화를 이루는 방법을 안내합니다.

### 3. 사주 성향 카드 (Saju Type Card)
- MBTI 성격 유형과 사주 정보를 결합하여 나만의 특별한 사주 성향 카드를 생성합니다.
- 생성된 카드는 이미지로 저장하거나 SNS에 공유할 수 있습니다.

### 4. AI 채팅 상담 (AI Chat Consultation)
- AI 챗봇과의 대화를 통해 구체적인 운세 고민이나 궁금한 점을 질문하고 답변을 받을 수 있습니다.
- 실시간 스트리밍 답변을 통해 자연스러운 상담 경험을 제공합니다.

### 5. 커뮤니티 (Community)
- 다른 사용자들과 운세 결과를 공유하고 서로의 고민을 나눌 수 있는 공간입니다.
- AI가 생성한 초안을 바탕으로 쉽게 게시글을 작성할 수 있습니다.

### 6. 프리미엄 사주 (Premium Saju)
- 더욱 깊이 있는 분석이 필요한 사용자를 위한 유료 심층 분석 리포트를 제공합니다.
